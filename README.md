# 程序员职场英语实战手册

面向中国软件工程师的职场英语训练指南。  
目标不是考试，也不是刷词汇量数字，而是在真实工作中建立可用英语能力：能说、能写、能评审、能汇报。

English readers can use: [README in English](README.en.md)

## 这份文档能帮你什么

- 让你快速建立“可开口、可落笔”的职场英语能力，而不是停留在输入阶段。
- 让你把英语学习和日常研发工作绑定，减少额外学习成本。
- 让你在文档、评审、汇报场景中，稳定输出清晰、可理解、可执行的表达。
- 给你一套可重复执行的方法：`repeat -> shadowing -> write -> review -> refine`。

## 你可能正遇到的困惑

- 我能看懂一些英文，但开口时卡住，写作时更卡。
- 我听了很多材料，听力还是不稳定，尤其是技术语境。
- 我知道背单词有用，但背完不会用，进步感很弱。
- 我写英文总担心不地道，担心中式表达被看出来。
- 我不知道每天该怎么练，练什么，怎么判断是否有效。

## 核心观点

- 说和写是主抓手：听力问题主要靠“说”突破，语法问题主要靠“写 + 反馈”解决。
- 不走脱离场景的死记硬背路线，词汇要在真实表达中积累。
- 不追求“完美英语”，先追求“稳定可懂、能协作”。

## FAQ

### 1. 需要先背 3000 个单词再开始吗？

不需要。  
先在真实工作场景里学词、用词，效率更高。

### 2. 我听不懂，是不是应该先狂刷听力？

不要只刷听力。  
听不懂常见根因是发音和输出能力不足，先补说。

### 3. 我口音重，能不能先不练口语？

不能。  
不要求像母语者，但必须保证可懂度和稳定性。

### 4. 我写出来全是中式英语怎么办？

正常。  
先保证可读，再通过 AI 的逐轮反馈持续修正。

### 5. 每天最少需要多久？

建议 60 分钟：  
30 分钟复读/跟读 + 20 分钟写作 + 10 分钟复盘。

### 6. 一周真的会有效果吗？

通常会。  
最先变化的是开口意愿、听力信心和基础写作可用性。

### 7. 我英语基础很差，能用这套吗？

能。  
材料要降级，控制每句生词不超过 2 个。

### 8. 只用 AI 能学好吗？

AI 能大幅提高反馈密度，但你必须亲自输出。  
只看不练，效果很差。

### 9. 什么时候开始大量听？

当你能稳定做复读、跟读、总结之后。  
否则大量输入往往只是在“灌耳音”。

### 10. 目标是考试，还适用吗？

部分适用。  
但考试是另一套体系，需要单独准备。
